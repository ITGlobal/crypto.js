Interface ISignatureApplet

Интерфейс криптографического апплета для Validata CSP. Требования к реализации:

  1. Вызов любого метода должен завершаться моментально (< 10мс) независимо от входных параметров. Сама же операция может занимать длительное время.
  2. Во время выполнения операции не должно происходить длительных синхронных вызовов из JavaScript. Это может приводить к "залипанию" UI.
  3. Длительные операции (> 1c) рекомендуется сопровождать уведомлениями о прогрессе {@link IPromise#progress}.
